# DeskMate Upgrade Changelog

## New Features

### 1. Enhanced Multi-tasking
- **YouTube Integration**: You can now say "Play RRR glimpse on YouTube" or "Search for Python tutorials on YouTube". DeskMate will directly open the search results or video.
- **Website Shortcuts**: Added support for more websites including university portals (Woxsen, IIT Hyderabad, GITAM) and AI tools (Perplexity, Grok).

### 2. File System Operations
- **Create Files**: Say "Create file notes.txt" to create a new file on your Desktop.
- **Create Folders**: Say "Create folder ProjectX" to create a new folder on your Desktop.
- **Smart Defaults**: Relative paths automatically default to your Desktop for easy access.

### 3. Improved File Explorer
- **Home Directory**: "Open File Explorer" now opens your Home directory by default, giving you access to all your folders (Documents, Downloads, Desktop, etc.).
- **Desktop Search**: Say "Search for report on Desktop" to specifically search your Desktop.

### 4. Terminal Integration
- **Open Terminal**: Say "Open terminal" to launch a new command prompt window rooted in your user directory.
- **Cross-Platform**: Works on Windows (CMD), macOS (Terminal), and Linux (GNOME Terminal).

## Technical Improvements
- **Robust Intent Parsing**: Rewrote the intent parser to better understand complex commands.
- **System Control Plugin**: Centralized system operations for better reliability and safety.
- **Conversational AI**: Maintained the ability to answer general questions and engage in chat.

## How to Use
- **Play Video**: "Open YouTube and play [video name]"
- **Create File**: "Create file [filename]"
- **Create Folder**: "Create folder [foldername]"
- **Open Terminal**: "Open terminal"
- **Search Desktop**: "Search for [filename] on desktop"
